# Elektrikli-Ara-arj-stasyonlar-nda-G-venlik-Analizi-BSG
Bu proje, elektrikli araç şarj istasyonlarında kullanılan OCPP (Open Charge Point Protocol) ve CAN Bus protokollerini tek bir simülasyon ortamında birleştirerek mesaj dönüşümü, güvenlik testi, MitM saldırı senaryoları ve vcan0 üzerinden CAN frame analizini gerçekleştirmeyi amaçlamaktadır.
